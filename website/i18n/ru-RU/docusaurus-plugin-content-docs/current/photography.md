### Конфиги GCAM

:::note
- «**Лучший**» — понятие субъективное. Попробуйте все доступные конфигурации и варианты, чтобы определить, какой из них лучше всего подходит именно вам!
- Для устройств на базе Snapdragon и MTK обычно рекомендуется использовать варианты **Snap** и **Aweme** соответственно.
- Установки только приложения GCAM недостаточно для получения наилучших результатов; именно файлы конфигурации, настроенные создателем конфига с учетом сенсора/объективов камеры устройства, делают его пригодным для использования и обеспечивают наилучшую производительность.
:::

#### Импорт конфигурации

1. Скачайте файл конфигурации (`.xml`) по ссылкам ниже.
2. Откройте загруженное приложение GCAM хотя бы один раз. Обязательно предоставьте доступ к файлам, хранилищу, камере и другие необходимые разрешения.
3. Папка для соответствующего варианта камеры будет создана автоматически в корне внутреннего хранилища. Путь может отличаться, но обычно он следующий:
- **AGC** → `Внутреннее хранилище/Download/AGC/AGC X.Y/configs/`  
- **LMC** → `Внутреннее хранилище/LMCX.Y/`  
- **SGCAM** → `Внутреннее хранилище/SGCAM/X.Y.Z/XML/`

> `X.Y` или `X.Y.Z` обозначает версию приложения (например, `9.2` или `9.2.114`).

4. Переместите загруженный файл конфигурации `.xml` или `.agc` по пути, указанному в шаге 3.
5. Снова откройте приложение GCAM и загрузите файл конфигурации следующим образом:
- Для AGC: значок «Еще настройки» > Предоставить разрешения на доступ ко всем файлам > вернуться > Перейдите в **Settings → Load Config**, затем выберите файл `.agc`, который вы сохранили в папке AGC Configs, и нажмите **Save**.
- Для LMC: дважды нажмите на пустую область между **кнопкой затвора** и **значком переключения камеры**, выберите конфиг и нажмите **Import**. Предоставьте разрешения, если появится соответствующий запрос.
6. Конфигурация будет применена автоматически. Переключайтесь между профилями (если они доступны) в верхней панели.
