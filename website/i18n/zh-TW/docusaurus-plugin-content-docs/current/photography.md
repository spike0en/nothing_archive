### GCAM 設定檔

:::note
- 「**最佳**」是主觀的。請嘗試所有可用的設定和變體，以確定最適合您的設定！
- 對於基於 Snapdragon 和 MTK 的裝置，通常建議分別使用 **Snap** 和 **Aweme** 變體。
- 僅安裝隨機版本的 GCAM 應用程式不足以獲得最佳效果；由設定檔創作者根據裝置的相機感光元件/鏡頭調整的設定檔才是使其可用並發揮最佳效能的關鍵。
:::

#### 匯入設定檔

1. 從下方連結下載設定檔（`.xml`）。
2. 打開下載的 GCAM 應用程式至少一次。請務必授予文件、儲存空間、相機和其他必要權限。
3. 相應相機變體的資料夾將自動在內部儲存空間的根目錄中建立。路徑可能會有所不同，但通常如下：
- **AGC** → `內部儲存空間/Download/AGC/AGC X.Y/configs/`  
- **LMC** → `內部儲存空間/LMCX.Y/`  
- **SGCAM** → `內部儲存空間/SGCAM/X.Y.Z/XML/`

> `X.Y` 或 `X.Y.Z` 代表應用程式版本（例如：`9.2` 或 `9.2.114`）。

4. 將下載的 `.xml` 或 `.agc` 設定檔移動到步驟 3 中提到的路徑。
5. 再次打開 GCAM 應用程式並按照以下步驟載入設定檔：
- 對於 AGC：更多設定圖示 > 授予所有檔案的存取權限 > 返回 > 前往 **Settings → Load Config**，然後選擇您保存在 AGC Configs 資料夾下的 `.agc` 檔案，然後點擊 **Save**。
- 對於 LMC：連按兩下**快門按鈕**和**相機切換圖示**之間的空白區域，選擇設定檔，然後點擊 **Import**。如果出現提示，請授予權限。
6. 設定檔將自動套用。您可以從頂端列切換不同的設定檔（如果可用）。
