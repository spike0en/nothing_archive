# Frogger-B4.1-260630-1839

## Update changelogs
What's New in This Update  

### Essential Voice Enhancements
🔡 Translation/transcription can now be triggered via double-clicking the Essential Key when using the keyboard. Set it up in Settings > Intelligence toolkit > Essential Voice > Customise Essential Key.  
⌨️ Quick switch between translation and transcription is now available by long-pressing the icon on the keyboard.  
🎙️ Optimised Essential Voice for a smoother performance.  

### Camera Enhancements
📹 Lock Lens keeps your video framing steady by preventing automatic lens switching during recording. Enable in Camera settings > Lock Lens.  
🏙️ Improved skin tone and colour accuracy for outdoor shots.  
🎬 Better shooting stability for smoother capture.  

### General Improvements and Bug Fixes
🎚️ Refined the volume slider to allow seamless and more precise adjustment when dragging.  
🎫 Optimised eSIM usage stability across more scenarios.  
⚙️ Enhanced overall system stability.  

## Notes
Your device may use more power and feel warmer during installation. Performance will return to normal after the update is complete.
