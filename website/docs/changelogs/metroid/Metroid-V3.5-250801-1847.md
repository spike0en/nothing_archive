# Metroid-V3.5-250801-1847

## Changelog

### What's New in This Update

### Glyph Interface

📸 A Glyph Matrix styled photo will now also be saved when capturing with the Glyph Mirror toy.  
🔔 Resolved an issue causing ringtones to be set to Classic when in Generative mode.  
📱 Resolved incorrect mode selection for ringtones after phone data transfer.  
📞 Fixed a bug with the Glyph Interface response when receiving Telegram calls.  

Camera Enhancements  
🤖 Added support for AI SuperRes Zoom between 20x and 30x.  
📸 Improved shutter response after continuous captures.  
🎨 Optimised autofocus and colour accuracy.  

General Improvements and Bug Fixes  
🎯 Fixed the issue where the default search engine option may not show up in the App drawer & Search settings page.  
🛠️ Fixed an issue causing the second character to go missing in Universal Search.  
🔲 Improved Essential Notification border UI.  
📄 Resolved the issue of cloned WhatsApp app unable to open PDF files.  
📱 Improved refresh rate for Snapchat for a smoother experience.  
🎥 Fixed stuttering when playing 4K 60FPS videos in Gallery.  
🌞 Enhanced Adaptive Brightness for smoother screen brightness transitions.  
🔌 Fixed occasional charging disconnections when using certain third party chargers.  
⚙️ Improved overall system stability.  
📝 Improved copywriting.  

## Notes:

The device may temporarily heat up and increase power consumption during the update and will quickly return to normal after the process is complete.
