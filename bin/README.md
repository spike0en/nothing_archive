# Third-party Tools (Binaries)

The files in this directory are third-party binaries used by the Nothing Archive project for extracting OTA packages. They are not owned by the Nothing Archive authors and are provided here for convenience.

## Tools Summary

### 1. OTA Extractor (`ota_extractor`)
- **Source**: [luk1337/oplus_archive](https://github.com/luk1337/oplus_archive)
- **License**: Apache License 2.0 / AOSP
- **Description**: A tool to extract payload.bin files from Android/Oryza/Oplus OTA packages.

### 2. Supporting Libraries (`.so` files)
- **Source**: Android Open Source Project (AOSP) / Chromium
- **License**: Apache License 2.0 / BSD-3-Clause
- **Description**: Standard shared libraries required by the `ota_extractor` move.
