# Tetris-B4.0-260225-1904

## Update changelogs:
What's New in This Update  

### New Features
🪀 Essential Apps now features a recommendation section in the widget library, providing quicker access to the Nothing Playground to discover AI-powered community creations.  

### General Improvements and Bug Fixes
🌌 Improved the animation smoothness of dragging icons on the Launcher.  
⛅ Optimised the Weather widget location permission prompt to reduce disruption.  
⚙️ Enhanced overall system stability.  
⛓️ Fixed an issue where the Speed and compatibility could not be switched in Wi-Fi hotspot settings.  
🎧 Fixed an issue causing call audio loss when switching between multiple headphones.  
🔈 Fixed an issue where headphone audio was lost when resetting the Bluetooth sample rate to default while using LHDC or LDAC.  
🎨 Fixed an issue where Expert mode presets reverted to the incorrect zoom level when re-entering.  
🛡️ Updated to March 2026 security patch.  

## Notes
Your device may use more power and feel warmer during installation. Performance will return to normal after the update is complete.
